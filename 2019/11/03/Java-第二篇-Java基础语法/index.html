1. 注释

编写程序时添加注释：调高程序的可读性(编程思路和程序作用)。添加注释是一种良好的编程习惯。

注释不会出现在字节码.class文件中(编译器会跳过注释)。

Java的注释分为三种：单行注释，多行注释，文档注释。

单行注释：以”//“开头，”//“后面的内容为注释。

多行注释：以”/*“开头，以”*/“结尾，”/*“和“*/”之间的内容为注释。有两种形式：空白处多行注释，行内任意位置注释。

文档注释：以”/**“开头，以”*/“结尾，”/**“和”*/“之间的内容为注释。

2. 标识符

标识符：给包、类、方法、变量起名字。

标识符命名规则：

  1. 以字母，下划线”_”，美元符号”$”开头，数字不能开头
  2. 不能包含空格，不可以使用关键字和保留字
  3. 严格区分大小写
  4. 见名知意
标识符 规则 示例
包 全部小写 xxxyyyzzz
常量 全部大写 + 下划线 XXX_YYY_ZZZ
类、接口 首字母大写 + 驼峰原则 XxxYyyZzz
方法、变量 驼峰原则 xxxYyyZzz

3. 变量

变量 内容
格式 变量类型 变量名 = 值
作用 保存数据
本质 可操作的存储空间（使用变量名访问）
要求 必须先声明，后使用；必须声明其数据类型（强类型语言）
三要素 变量名，变量类型，作用域

基本数据类型8种：byte，short，int，long，float，double，char，boolean。引用数据类型3种：类，接口，数组。

3.3 类型转换

  1. 类型转换分为自动类型转换和强制类型转换
  2. 自动类型转换：容量小的数据类型自动转换成容量大的数据类型
  3. 强制类型转换：容量大的数据类型强制转换成容量小的数据类型，可能损失精度或溢出
  4. 强制类型转换格式：(类型) 变量
  5. 多种类型混合计算时，系统首先将所有数据转化成容量大的，再进行计算
  6. boolean类型不可以转化成其他类型

5. 运算符

5.1 算术运算符

运算符 说明 举例
+ 加 a + b
- 减 a - b
* 乘 a * b
/ 除 a / b
% 取模 a % b
++ 自增 a++
– 自减 a–

注意：

  1. 对于a % b = c，c的正负只取决于a的正负。即：-3 % -2 = -1；-3 % 2 = -1；3 % -2 = 1。
  2. 对于b = a++，a先赋值再自增；对于b = ++a，a先自增再赋值。
  3. 对于b = a–，a先赋值再自减；对于b = –a，a先自减再赋值。
  4. 对于整型运算：两个操作数只要有一个是long，结果为long；不存在long，结果为int，即使操作数是byte，short。
  5. 对于浮点型运算：两个操作数必须同时是float，结果才为float；只要有一个是double，结果为double。

5.2 赋值运算符

运算符 举例 等效表达式
= a = b —-
+= a += b a = a + b
-= a -= b a = a - b
*= a *= b a = a * b
/= a /= b a = a / b
%= a %= b a = a % b

注意：

  1. “=”是赋值运算符，把右边的值赋给左边的变量，实质是把右边的地址值赋给左边。
  2. 赋值运算符里面的b是表达式。

5.3 比较运算符

运算符 说明 举例
== 等于 a == b
!= 不等于 a != b
< 小于 a < b
> 大于 a > b
<= 小于或等于 a <= b
>= 大于或等于 a >= b

注意：

  1. 比较运算符的结果是boolean：true/false。
  2. “==”，”!=”：基本数据类型和引用数据类型都可以使用。
  3. “<”，”>”，”<=”，”>=”：仅数值型和字符型可以使用。

5.4 逻辑运算符

运算符 说明 举例
& 与（只要有一个false，结果为false） a & b
| 或（只要有一个true，结果为true） a | b
! 非（取反） !a
&& 短路与（只要有一个false，直接返回false） a && b
|| 短路或（只要有一个true，直接返回为true） a || b
^ 异或（相同为false，不同为true） a ^ b

注意：

  1. 逻辑运算符的操作数和结果都是boolean：true/false。
  2. 数学中的 3 < x < 6，Java中要写成：x > 3 & x < 6。
  3. “&”和”&&”的区别：”&”左右两边都需要计算；”&&”如果左边true，右边参与运算，如果左边false，右边不参与计算，直接返回false。
  4. “|”和”||”的区别：”|”左右两边都需要计算；”||”如果左边true，右边不参与运算，直接返回true，如果左边false，右边参与计算。
  5. “^”：对于任何数a，a ^ a = 0，a ^ 0 = a。

真值表：

a b a & b a && b a | b a || b ! a a ^ b
true true true true true true false false
true false false false true true false true
false true false false true true true true
false false false false false false true false

5.5 位运算符

运算符 说明 举例
& 与 a & b
| 或 a | b
~ 取反 ~a
^ 异或 a ^ b
<< 左移 a << b
>> 右移 a >> b

注意：

  1. 位运算符是对左右两个数进行二进制位的运算。
  2. 在不越界的情况下：”<< 1”相当于”*2”，”>> 1”相当于”/2取商”。
  3. 关于数字的算法题中，经常通过位运算进行骚操作。

运算符 说明
x ? y : z x为boolean；x为true，结果为表达式y；x为false，结果为表达式z

注意：

  1. 三元运算符是”if…else…”的简写形式。

优先级 运算符
1 ()

3 *，/，%
4 +(加法)，-(减法)

9 &

注意：

  1. 运算符的优先级比较难记住，规避使用混合运算符，优先使用”()”。
  2. 混合运算符一般出现在面试题里，如果程序开发中大量使用，会被开除。
  3. 逻辑与，逻辑或，逻辑非的优先级需要掌握：! > & > |。

6. 流程控制语句

流程控制语句分为3类：顺序结构，选择结构和循环结构。

6.1 顺序结构

顺序结构：程序从上到下逐行执行，中间没有任何判断和跳转。

6.2 选择结构

选择结构：根据判断条件，选择性的执行某段代码，分为”if…else…”和”switch…case…”。

if if-else if-else if-else switch-case
if（布尔表达式）{
  语句块
}		 if（布尔表达式）{
  语句块1
} else {
  语句块2
}		 if（布尔表达式1）{
  语句块1
} else if （布尔表达式2）{
  语句块2
} else {
  语句块3
}		 switch（表达式）{
  case 值1：
    语句1
    [break]
  case 值2：
    语句2
    [break]
  [default：
    语句3
  ]
}

注意：

  1. if语句：对布尔表达式进行判断，结果为true，则执行语句块，结果为false，则跳过语句块。
  2. if-else语句：对布尔表达式进行判断，结果为true，则执行语句块1，结果为false，则执行语句块2。
  3. if-else if-else语句：对布尔表达式1进行判断，结果为true，则执行语句块1；否则，对布尔表达式2进行判断，结果为true，则执行语句块2；否则执行语句块3。
  4. switch-case语句：根据表达式的值匹配case标签，执行相应语句。如果表达式的值与任一case标签都不匹配，执行default语句。”[]”内的break和default可以去掉。
  5. switch-case语句：用于多值的判断，区间的判断不行。
  6. switch-case语句：表达式的值只能是以下6种类型（byte，short，int，char，Enum，String）。
  7. switch-case语句：case里面如果没有break，程序会按照顺序结构执行每一个case。

6.3 循环结构

循环结构：根据循环条件，重复性的执行某段代码，分为”while”，”do…while…”和”for”。

while do-while for
  初始表达式
while（布尔表达式）{
  循环体
  迭代因子
}		   初始表达式
do {
  循环体
  迭代因子
} while（布尔表达式）		 for（初始表达式;布尔表达式;迭代因子）{
  循环体
}

注意：

  1. while语句：对布尔表达式进行判断，结果为true，则执行循环体，结果为false，则跳过循环体。
  2. do-while语句：先执行一次循环体，然后对布尔表达式进行判断，结果为true，则执行循环体，结果为false，则跳过循环体。
  3. for语句：初始表达式用来初始化循环变量；迭代因子用来控制循环变量的增减；对布尔表达式进行判断，结果为true，则执行循环体，结果为false，则跳过循环体。

嵌套循环：在一个循环语句内部嵌套一个或多个循环语句。嵌套循环执行次数 = 外层循环次数 * 内层循环次数。外层循环执行一次，内层循环执行一轮。外层循环控制行，内层循环控制列。

死循环：循环条件一直满足。退出死循环的方式：循环条件变成false；使用break语句。

6.4 跳转结构

跳转结构：用于结束方法或循环，分为”break”，”continue”和”return”。

语句 使用范围 作用
break 循环结构，switch-case结构 结束整个循环
continue 循环结构 结束本次循环
return 方法 结束整个方法

注意：

  1. “break”，”continue”和”return”用于跳转，后面不可以添加其他语句，因为不会被执行。
